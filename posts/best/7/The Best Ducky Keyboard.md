## FAQ


### What is a Ducky Keyboard?

A Ducky Keyboard is a special type of USB keyboard designed to execute custom scripts when specific keys are pressed. Unlike standard keyboards, Ducky Keys use unique circuitry to interpret specially programmed commands, allowing users to automate tasks or perform complex operations with just a few keystrokes. 

The concept behind Ducky Keyboards stems from USB HID (Human Interface Device) capabilities, enabling them to be recognized by computers as standard keyboards while executing custom functions through firmware modifications. 


### What makes a Ducky Keyboard different from a regular keyboard?

Regular keyboards function through basic electrical signals that tell your computer what key was pressed. Ducky Keyboards, however, include custom firmware which allows them to interpret specific key combinations as commands to run scripts or execute predefined actions. 

While regular keyboards have limited functionality beyond basic input, Ducky Keyboards can perform tasks like automating mouse clicks, launching applications, or performing series of keystrokes - making them particularly useful for gaming, productivity, or scripting tasks. 


### How do I program a Ducky Keyboard?

Designing scripts for Ducky Keyboards requires basic knowledge of programming and understanding of what key codes correspond to specific keystrokes. Users typically use specialized software to upload these custom scripts through the keyboard's USB port.
