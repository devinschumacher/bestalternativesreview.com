## FAQ

### What is Goxlr?

Goxlr is a comprehensive broadcasting solution that combines a USB microphone/preamp, audio interface, and software for live streaming and podcasting. It's designed to simplify audio recording and broadcasting for YouTubers, streamers, and podcasters. 

The system includes the Goxlr Core hardware unit, which processes audio inputs and connects to your computer via USB, along with the Goxlr Software, which enables broadcasting to various platforms including YouTube, Twitch, and Facebook Live. 


### Which platforms does Goxlr support?

Goxlr supports multiple streaming and broadcasting platforms, including YouTube, Twitch, Facebook Live, and Periscope. The software allows you to select your preferred destination and stream directly from the Goxlr Core hardware. 

The system also includes support for audio-only broadcasting, allowing you to stream to services like SoundCloud and Radio Garden with high-quality audio. 

![Goxlr-5](https://imagedelivery.net/vy2bglCGN6hEeWOnSe2c7A/Goxlr-5/h=540,fit=pad,background=black)


### What are the technical specifications?

The Goxlr Core features a 20-channel audio mixer with independent gain controls and phantom power for condenser microphones. It supports up to 24-bit/96kHz audio quality and has three simultaneous audio outputs (USB, HDMI, and Line Out). 

The included software requires a 64-bit operating system and supports Windows 10, macOS Catalina, and later versions of both operating systems. 


### What connectivity options are available?

The Goxlr Core connects to your computer via USB-C, providing reliable data transfer for both audio and control signals. The system also includes a 3.5mm headphone jack for monitoring, and the USB-C port powers the device. 

Additional connectivity options include three 3.5mm inputs (two XLR/TRS combo inputs and one 1/4" instrument input), a stereo output for headphones or powered speakers, and an HDMI pass-through port for video capture. 

![Goxlr-6](https://imagedelivery.net/vy2bglCGN6hEeWOnSe2c7A/Goxlr-6/h=540,fit=pad,background=black)


### What about software compatibility?

The Goxlr Software is compatible with both 64-bit Windows and macOS systems, including the latest versions of each operating system. It requires system resources equivalent to a modern notebook computer with 4GB RAM and an Intel Core i5 processor or higher. 

Additional software tools are available through the Goxlr website, including the Goxlr Mixer for on-screen control and the Goxlr Capture software for recording and editing audio content. 


### How do I get started with Goxlr?

Setting up Goxlr is straightforward. Begin by connecting the Goxlr Core to your computer via USB-C and downloading/installing the Goxlr Software. The software walks you through basic configuration, including selecting your broadcast platform and setting up audio inputs. 

For live streaming, you'll need to create an account on your chosen platform and obtain the necessary stream keys or settings. The Goxlr software integrates these settings directly, making your broadcast setup simple and efficient.
